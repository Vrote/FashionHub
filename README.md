Overview:

FashionHub is a full-stack e-commerce fashion website inspired by platforms like Flipkart. It provides a smooth online shopping experience with multiple web pages for browsing products, viewing details, and managing user data.

Features:

Multi-page fashion e-commerce website

Product listing and product detail pages

User authentication & data management

Backend APIs using Node.js

Database integration for products and users

Responsive UI using HTML & CSS<br><br>

Tech Stack:

Frontend: HTML, CSS

Backend: Node.js

Databases: MongoDB, MySQL

Version Control: Git<br><br>


How to Run:<br>
npm install<br>
